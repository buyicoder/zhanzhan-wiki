> 从「不敢用 AI」到「拥有自己的个人复利成长系统」。本书为 Windows 版，MacBook 用户请使用 Mac 版手册。
> 你不需要成为程序员，不需要人工智能博士学位。你需要的只是这本手册、一台电脑，和每天一点点时间。

## 这本手册怎么用

**1. 跟着里程碑走，别跳。** 全课共 8 个大里程碑，每个里程碑由若干任务组成。每个任务都有明确的「🎯 成功标志」——达到了才算完成，没达到就停下来解决，不要往下冲。

**2. 认识手册里的符号。** 全书统一使用这套记号：

| 符号 | 含义 |
|------|------|
| 🎯 成功标志 | 这一步做对了长什么样。这是唯一的验收标准 |
| 📸 截图 | 此处有配图，图上是你应该看到的画面 |
| 🔗 快捷网址 | 直接点击，不用自己搜 |
| 💡 温馨提示 | 前人踩过的坑、更顺手的做法 |
| ⚠️ 注意 | 不看会出问题的警告 |
| 🔍 刚才发生了什么 | 操作完成后的原理解释——知其然，也知其所以然 |
| 🚀 快速通道 | 只想往前走？看这个就够了 |
| 🔬 深入一层 | 有好奇心？这里给你多讲一点 |

**3. 可浅可深，自己选。** 每个任务的正文只包含「必须做的操作」。标着 🔬 的部分全部可以跳过，不影响毕业；但每多读一个 🔬，你对电脑和 AI 的理解就厚一层。第一遍学建议全走 🚀，毕业后回头再读 🔬。

**4. 卡住了怎么办（重要）。** 本课的核心心法：**遇到任何问题，先问你手边的 Agent。** 把报错原文、你在做哪一步、你期望什么结果，一起发给它。这本身就是课程要训练的能力。手册里每个任务的 ⚠️ 部分列出了最常见的卡点。

## 课程总路线

```text
里程碑一   让 Agent 第一次操作你的电脑        （建立体感：AI 能动真格的）
里程碑二   准备可持续使用的 AI 环境           （工具链：WorkBuddy / Codex / Claude）
里程碑三   理解 AI 是怎样操作电脑的           （概念地基：LLM、程序、副作用、命令行）
里程碑四   用 Git 给 Agent 的修改建立存档     （安全网：随时可以后悔）
里程碑五   用 GitHub 保存项目和参与协作       （资产库：项目上云、需求驱动开发）
里程碑六   毕业项目：指挥 Agent 完成贪吃蛇     （综合实战：一个真正能玩的游戏）
里程碑七   发布上线：让别人真正访问你的作品    （从作品到产品的第一步）
里程碑八   个人复利成长系统                  （从学会 AI 到让 AI 认识你）
```

**毕业标志：** 独立完成一个可运行、可测试、可回溯版本、可在线访问的贪吃蛇游戏，并搭建起自己的记录-沉淀-复用循环。
