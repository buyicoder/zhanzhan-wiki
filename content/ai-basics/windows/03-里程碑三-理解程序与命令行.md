## 这一关解决什么问题

Agent 会替你敲命令，但如果你完全不知道程序、参数、文件和副作用是什么，就无法判断它做得对不对。这一关不是把小白训练成程序员，而是补上最低限度的计算机直觉。

## 你将完成什么

- 用“输入 → 黑箱 → 输出”理解 LLM 与普通程序的共同点和差异。
- 运行一个只做计算的程序，再观察一个会创建文件的程序。
- 用终端完成移动目录、查看文件和运行脚本等基础动作。

## 成功标准

你能用自己的话解释程序为什么会改变电脑状态，能看懂 Agent 准备执行的命令大致影响哪里，并能独立运行一个简单 Python 程序和一个批处理脚本。

## 能力迁移

以后 Agent 报错时，你会先区分是命令、路径、权限还是程序逻辑问题；面对陌生工具，也能从输入、输出和副作用判断风险。
