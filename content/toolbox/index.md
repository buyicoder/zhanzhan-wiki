## 我的信息源

- AI 资讯：AI Hot 这类聚合站，收集全球 AI 资讯，拿到第一手资料
- GitHub：有一些项目专门收集新星宝藏项目
- 公众号，个人 Agent 经验分享，文字永远效率最高：赛博禅心、数字生命卡兹克
- 抖音，把握大众情绪。你可以不屑于看抖音，但是你其实需要做抖音：数字游牧人
- 展会、沙龙、企业、咨询：深入业务了解真实需求
- AI 本身：主动收集信息的最佳方式。任何你想知道的方向，对着 AI 倾注注意力，就可以得到你想要的
- 官方文档：新工具的最好教程
- GitHub issue、PR 和维护者反馈
- arXiv 与论文
- 用户访谈、客户现场和真实业务数据
- 电影、番剧、小说等叙事模型
- 自己的项目日志和失败记录

## 我的工具箱

- Typora：好用的 Markdown 编辑器
- Obsidian：双链本地 wiki
- 番茄 todo：时间管理，P 人强迫自己用 J 人的方法
- Git：版本管理必备
- GitHub：项目管理、信息源、合作平台、高质量社交平台、对外名片、部署个人网站，一站全包
- GitHub CLI：便于自动化
- Agent 工具
  - Codex：对新手友好
  - Claude Code：最强 Agent，但国内用上有门槛
  - Cursor：不用特殊网络环境也能用，切换模型方便，小步前进式开发，控制力强
  - VS Code：生态强大，有免费 Agent 可用
  - WorkBuddy：速度快，下载方便，国内生态强，AI 回复可视化做得好，专家调用好用，每日免费额度很香，适合小白上手
- Jarvis：自制个人 AI 管家，了解我的几乎全部信息，帮我制定日程、梳理创业思路
- StarPeer：自制 GitHub 项目狩猎机，寻找高价值项目，提交简单 PR 刷好感度，多次 merge 后提醒我深度参与
- Jarvis 主控-负责人系统：我不在窗口之间切换，只看 Jarvis 主控窗口。它收到指令后分派给对应负责人线程，负责人完成后发消息回主控，主控向我汇报。一个 Jarvis 帮我管理所有线程，它也因此清楚我一天做的所有事，便于总结复盘
- 飞书：知识库和多维表格很好用，便于分享，企业 AI 转型会用到
- 飞书 CLI：便于 AI 操控自动化
- 搜索引擎：Bing 或 Google。不要使用广告太多的搜索工具，一次广告的干扰可能只有几秒钟，累计起来是一笔巨额时间
- Playwright：模拟浏览器操作
- Computer use：模拟鼠标操作
- Docker：部署容器
- Remotion：自动化剪辑
- 剪映：剪辑

## 我的工作方法

- 番茄工作法：25 分钟一个番茄，番茄内不看手机不接收消息，只做一件事；25 分钟结束立即停止，休息 5 分钟，不允许继续工作
- 目标 → 成功标准 → 调查 → 计划 → 执行 → 测试 → 反馈 → 沉淀
- 先在 GitHub 寻找成熟方案，不重复造轮子
- 总控 + 项目负责人机制
- Maker + Checker
- 小步执行，保留回滚点
- 用真实浏览器和截图验收（需要 Agent 有视觉能力）
- 区分原型可用、内部试用、外部上线
- AI 负责发散，人负责收敛于判断
- 先描述需求与成功标准，再让 Agent 工作

## 我的 AI 使用心得

- 使用 Agent 而不是聊天 AI
- 努力用上最先进的 Agent，先尝试一个月
- 使用一个窗口管理所有窗口，不要频繁切换
- 明确目标，不要为了用 Agent 而用 Agent，不要为了获取 skill 而获取 skill
- 使用 AI 模拟理想中未来普通的一天，用于祛魅，感知当下的力量
- 明确目标，调查，制定计划，执行，测试，反馈，沉淀经验，如此循环
- 让 Agent 开始之前，先去 GitHub 看是否已有类似项目
- AI 时代发散思维贬值，收敛能力升值
- AI 无法批量生产可变现的娱乐产品，包括小说、动漫、游戏
- AI 的主战场在工作和学习，不是娱乐
- AI 无法稳定实现使用者能力范围以外的事情，也无法实现使用者认知范围以外的事情
- AI 加速使用者实现能力范围以内的事情，加速使用者扩大认知范围，也就是学习效率

## 其它能力获取方式

- **英语**：100LS 方法，反复观看同一部无字幕电影可大幅提升听说，反复阅读同一本英文爽文可大幅提升阅读，或者刷英文帖子。冲着迅速提分去，请用传统应试办法：刷原题
- **表达能力**：发视频，先做起来。解决心理问题：不要把自己看得太重要，这个世界上没有多少人真正在乎你做了什么，这件事和你是谁无关。发视频还能训练元认知
- **爆款口播能力**：真诚表达 + 露脸 + 适配音乐。文案写出来之后就知道能不能爆，但只要写出来了就拍、就剪、就发，不要因为写不出好文案就不发。没火的视频也是账号的重要组成部分
- **元认知能力**：把自己当作另外一个人冷静观察的能力。极大提升复盘和反思，避免被情绪左右。冥想、阅读、写作、拍视频、和元认知强的人聊天，都可以提升
- **赚钱能力**：找到一个能卖的东西 + 获客 + 交付。上班是把综合能力加劳动时间打包出卖；做生意，卖什么都可以
- **投资能力**：意识到钱可以生钱。资产等于能生钱的东西，负债等于能吞钱的东西。尽可能购入资产，减少负债
